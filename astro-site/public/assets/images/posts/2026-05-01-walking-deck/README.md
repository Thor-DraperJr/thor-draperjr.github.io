# Walking Deck Image Slots

Put walking deck photos in this folder. The component checks for these exact filenames at build time and replaces each placeholder automatically when the file exists.

| File | Used for | Suggested image |
| --- | --- | --- |
| `hero-portrait.jpg` | Cover portrait | Professional photo or favorite candid that feels like you |
| `charlotte-uptown.jpg` | Charlotte home base | Uptown, Belk Theater, city view, or Charlotte moment |
| `banff-wedding.jpg` | Personal story | Banff wedding photo tied to the 7/11 story |
| `lloyd.jpg` | Personal texture | Lloyd photo with personality |
| `operator-years.jpg` | Career proof | Enterprise, gym ownership, sales floor, or early operator chapter |
| `teaching-tech.jpg` | Technical/teaching proof | Bootcamp teaching, Microsoft event, certification moment, or technical community photo |

Use `.jpg` with the exact names above unless you update the paths in `src/components/WalkingDeck.astro`.
